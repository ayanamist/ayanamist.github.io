玩aircrack-ng，要打开网卡的monitor模式，从他家wiki到网上教程，全都是用airmon-ng来打开，于是在这个bcm4313芯片上全部失败，因为只有brcmsmac才支持设备是wlan0，但信号超弱，用sta驱动时设备是eth1，虽然信号好但直接提示设备不支持……

后来闲来无事翻broadcom的readme，发现原来他家STA驱动非常良心的带了monitor模式，只不过开启方式比较非主流。

HOW TO USE MONITOR MODE
———————–
To enable monitor mode:
$ echo 1 > /proc/brcm_monitor0
 
Enabling monitor mode will create a ‘prism0’ network interface. Wireshark and other netwokk tools can use this new prism0 interface.
 
To disable monitor mode:
$ echo 0 > /proc/brcm_monitor0

连airmon-ng都不需要，直接prism0替换mon0继续往下走就可以了
